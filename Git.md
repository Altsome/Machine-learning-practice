# Git = Push & Pull

---
## Push = Upload
    add -> commit -> push
### 1.add
    git add.: 곁에 있던 모든 변경사항들을 commit에 저장
### 2.commmit
    git commit -m "커밋메시지": 명시적으로 commit을 남김
### 3.push
    git push
---
## Pull
    git pull
---
## Clone
    git clone 주소
---
## Git command
### 1.git status
    add 해야할 변경사항들을 제시
### 2.git config
    설정 변경
    user.name "이름": 유저 이름 변경
    user.email "이메일": 유저 이메일 변경
### 3.git log
    log 제시
